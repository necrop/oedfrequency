Introduction

The OED Frequency site displays diachronic word-frequency data for OED entries and subentries, covering the period 1750-2010. Frequency data is derived mainly from the Google Ngrams data set, revised and adjusted to handle homographs, different parts of speech, etc. For more information on the way the data is derived, see the various background documents in the Documents page.

Frequency data is not stored for obsolete entries and subentries.

Quick search

For Quick Search, use the input box on the right of the top navigation bar. Quick Search returns a list of lemmas (entries and subentries) matching the query term(s). Results are returned in alphabetical order, by lemma.

Searching is case-, accent-, space-, and hyphen-insensitive. Several extended search methods are supported:

  • Use '*' as a wildcard, e.g. 'kn*' returns all lemmas beginning 'kn...';
  • Use the underscore ('_') to express a range, e.g. 'mo_muzz' returns all lemmas between 'mo' and 'muzz';
  • To search for several different lemmas, use commas as separators, e.g. 'bread, butter, cheese'. (This can be useful for comparing frequencies.)

Advanced search

The Lemma field operates just like Quick Search input box, and supports the same extended search methods. The Lemma field may be left blank, in which case it will be treated as 'a_zzz', i.e. all lemmas are returned.

The Entry headword field is usually left blank, but can be used to restrict results to lemmas found within a particular entry. For example, to see which are the highest-frequency subentries in mountain, n., leave the Lemma field blank, put 'mountain' in the Entry headword field, and select Sort by: frequency (descending).

The Entry type field allows you to select main entries or subentries.

The Wordclass field allows you to select lemmas with a particular part of speech. In the case of subentries, this is the part of speech of the lemma itself, not of its parent entry.

The Frequency band fields filter results to those within a given range of frequency bands, from 1 (very high-frequency lemmas) to 8 (very low-frequency lemmas). Bands are logarithmic: lemmas in band 1 are about 10x as frequent as lemmas in band 2, etc.

The Year field adjusts how frequency is evaluated. For example, if you use the Frequency band fields to specify that you want lemmas in bands 1 and 2 only, and use the Sort by field to sort results in frequency order, this will usually be evaluated with reference to lemma frequencies in the year 2000. But you can adjust this so that lemmas are evaluated with reference to their frequencies in, say, 1780.

Lemmas which are obsolete or not yet coined at the date specified in the Year field, or lemmas which fall outside the frequency range specified, are usually omitted from the results list. But you can include them by ticking the relevant Ghosts checkbox. 'Ghost' lemmas appear greyed out in the results list.

By default, results are returned in alphabetical order, by lemma (same as Quick Search). The Sort by selector allows you to choose a different order. For example, to see the highest-frequency evo... lemmas, enter 'evo*' in the Lemma field, and select Sort by: frequency (descending).

Sort order

By default, results for both Quick Search and Advanced Search are returned in alphabetical order, by lemma. At the top of the results list is a row of alternative sort orders, which you can click to change the way that the current results set is sorted.

Sort order can also be set in advance, by using the Sort by dropdown in the Advanced Search form.

If a particular date has been specified using the Year field in the Advanced Search form, frequency-based sort orders are computed with reference to each lemma's frequency at that time. Otherwise, frequency-based sort orders are computed with reference to each lemma's average frequency in the modern period (roughly, 1970-2010).

When sorting by increase or decrease since 1800, lemmas which have null frequency at 1800 or now are omitted (since in such cases change in frequency cannot be meaningfully calculated as a ratio). This means that lemmas which did not yet exist in 1800, or lemmas which have become obsolete since 1800, will be omitted.

Lemma detail

Click the lemma in a results list to go to the lemma detail page. This includes a chart showing frequency over time for all the individual forms and variant spellings making up the lemma (e.g. singular and plural in the case of a noun), and for the lemma as a whole. If the lemma is a main entry, it will also include a chart showing the position of this entry in a list of OED entries ranked by frequency.

Mouse over data points in the charts for more information.

The frequency table at the bottom of the page gives the complete set of raw frequency data for the lemma.

Frequency comparison

There are several different ways to generate charts comparing frequencies for different lemmas:

  • If a results list consist of 10 results or fewer, you can compare these directly by clicking 'Compare frequencies for these results' at the end of the list.
  • You can search for two or more lemmas with a comma-separated list in Quick Search, e.g. 'basketball,baseball', then click 'Compare frequencies for these results' at the end of the results list.
  • From the Lemma Detail page, you can compare the current lemma with any other recently-viewed lemma by clicking the relevant lemma listed in the 'Compare' section.

The Comparison page shows a chart in which each lemma being compared is shown as a separate data series. Click on a data points for more information.

You can remove lemmas from the chart by clicking the 'remove' button in the list of lemmas currently being compared. You can also add series for other recently-viewed lemmas.
